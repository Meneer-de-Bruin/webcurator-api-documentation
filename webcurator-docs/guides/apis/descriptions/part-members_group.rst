-----------------
**part: members**
-----------------
A list of members that belong to a group. This contains the following information:

==== ====== ========
**members**
--------------------
id   Number Required
type String Required
name String Required
==== ====== ========

| **id**
This is either the group or target identifier.

| **type**
This can be either 'group' or 'target'.
