| **state**
| The state of a target is an integer with the following values:

========= ===============
**State** **Description**
--------- ---------------
  1       Pending
  2       Reinstated
  3       Nominated
  4       Rejected
  5       Approved
  6       Cancelled
  7       Completed
========= ===============

The list of possible state values for targets can be retrieved with API :doc:`/guides/apis/api-target_states_GET`.
